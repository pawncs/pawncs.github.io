    Git提交三部曲

  • git add

    git add .命令效果与之前相同，此处-A指all.

  • git commit

    使用git commit -am "本次修改备注"来合并之前的步骤

  • git push

    第一次提交到本分支，origin是远程仓库默认名称，master是分支名称
    第2-n次提交到相同分支时，可简化为git push;同时，若仓库和本地不一致，应先执行git pull
